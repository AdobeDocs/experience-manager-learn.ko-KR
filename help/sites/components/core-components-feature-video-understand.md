# 핵심 구성 요소 이해{#understanding-core-components}

AEM 코어 구성 요소는 Adobe Experience Manager과 함께 사용할 표준 세트 구성 요소입니다. Adobe의 모범 사례와 표준을 기반으로 구축된 핵심 구성 요소는 모든 사이트 구현에 대한 기본 기능 세트를 제공합니다.
