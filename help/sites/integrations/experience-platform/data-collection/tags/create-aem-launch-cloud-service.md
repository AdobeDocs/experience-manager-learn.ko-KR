# AEM에서 Launch Cloud Service 구성 만들기 {#create-launch-cloud-service}

>[!NOTE]
>
>데이터 수집 기술 세트로 Adobe Experience Platform Launch 이름을 바꾸는 프로세스는 AEM 제품 UI, 콘텐츠 및 설명서에서 구현되고 있으므로 Launch라는 용어가 계속 여기에서 사용됩니다.
