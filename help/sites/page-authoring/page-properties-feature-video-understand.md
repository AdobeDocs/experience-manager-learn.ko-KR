# 페이지 속성 이해 {#understanding-page-properties-aem-sites}

다음 비디오에서는 Adobe Experience Manager의 페이지와 관련된 페이지 속성을 소개합니다.

>[!VIDEO](https://video.tv.adobe.com/v/32192?quality=12&learn=on)

이 비디오에서는 다음 페이지 속성 요소를 다룹니다.

* 기본
   * 제목 및 태그
      * 제목
      * 태그
      * 탐색 시 숨김
   * 기타 제목 및 설명
      * 페이지 제목
      * 탐색 제목
      * 소제목
      * 설명
   * 설정/해제 시간
   * 별칭 URL
* 고급
   * 설정
      * 언어
      * 리디렉션
      * 디자인
      * 별칭
   * 클라우드 구성
   * 템플릿 설정
   * 인증 요구 사항
   * 내보내기
* 썸네일
* 소셜 미디어 공유
* Cloud Services 구성
* 개인화
   * ContextHub 구성
   * 타깃팅 구성
* 권한
