# 페이지 편집기 사용 {#using-page-editor-with-aem-sites}

AEM Site의 페이지 편집기는 웹 콘텐츠를 만들고 편집할 수 있는 강력한 도구입니다. 페이지 편집기의 도구 및 기능은 페이지 편집기의 UI 도구 모음에서 액세스할 수 있습니다.

다음 비디오는 페이지 편집기의 주요 기능을 강조 표시합니다.

>[!VIDEO](https://video.tv.adobe.com/v/32011?quality=12&learn=on)

페이지 편집기의 기능은 다음과 같습니다.

* 구성 요소 및 에셋 드래그 앤 드롭
* 페이지의 모든 구성 요소에 대한 인덱스를 제공하는 콘텐츠 트리 패널
* 페이지 정보 패널을 사용하면 콘텐츠 작성자에 대한 추가 정보 및 빠른 작업에 액세스할 수 있습니다
   * 속성 열기
   * 페이지 롤아웃
   * 워크플로 시작
   * 페이지 잠금/페이지 잠금 해제
   * 페이지 게시/게시 취소
   * 템플릿 편집
   * 게시됨으로 보기
   * 관리자로 보기
   * 도움말
* 구성 요소 패널에 구성 요소 아이콘 및 설명 추가
* DAM에서 에셋을 편집하기 위한 직접 링크
* 인라인 콘텐츠 편집용 에뮬레이터와 응답형 그리드의 구성 요소를 보다 빠르게 관리할 수 있습니다.
* 페이지 편집 모드
   * 편집 모드
   * 레이아웃 모드
   * 개발자 모드
* 경로 탐색 양식 필드에 대한 자동 완성 및 키워드 검색
* 현재 페이지의 워크플로 상태 표시줄
