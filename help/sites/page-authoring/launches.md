# 론치

AEM Sites의 론치는 향후 릴리스에 사용할 웹 사이트 컨텐츠를 만들고, 작성하고, 검토할 수 있는 방법을 제공합니다. 론치를 만드는 동안 프로덕션 웹 사이트는 평소대로 계속 진화하고 매일 변경할 수 있습니다.
