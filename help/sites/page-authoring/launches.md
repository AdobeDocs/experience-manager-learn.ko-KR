# 론치

AEM Sites의 론치는 향후 릴리스를 위해 웹 사이트 컨텐츠를 작성, 작성 및 검토하는 방법을 제공합니다. 론치를 만드는 동안 프로덕션 웹 사이트는 일반적으로 발전하고 날로 변경될 수 있습니다.
