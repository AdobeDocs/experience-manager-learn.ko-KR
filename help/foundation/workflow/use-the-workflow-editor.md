# 워크플로우 편집기 사용

워크플로를 사용하면 Experience Manager에서 비즈니스 프로세스를 관리할 수 있으며, 컨텐츠를 자동으로 처리하고 인간의 의사 결정이 필요한 거버넌스 및 프로세스를 촉진하는 데 사용됩니다. 워크플로우는 일련의 단계로 구성되어 에서 생성 및 관리되는 워크플로 모델을 통해 정의됩니다 [!UICONTROL 워크플로우 모델 편집기].
