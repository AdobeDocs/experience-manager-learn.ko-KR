# 워크플로우 편집기 사용

워크플로우는 Experience Manager에서 비즈니스 프로세스 관리를 가능하게 하며, 컨텐츠를 자동으로 처리하고 인간의 의사 결정을 필요로 하는 관리 및 프로세스를 촉진하는 데 사용됩니다. 워크플로우는 일련의 단계로 구성되고 [!UICONTROL 워크플로우 모델 편집기]에서 만들고 관리하는 워크플로우 모델을 통해 정의됩니다.
