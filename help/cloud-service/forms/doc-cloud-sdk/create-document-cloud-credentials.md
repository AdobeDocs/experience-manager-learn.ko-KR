# Adobe I/O 프로젝트 만들기

1. 통합을 만드는 첫 번째 단계는 를 만드는 것입니다. [Adobe ID](https://account.adobe.com/)
1. 로 이동 [Adobe Developer 콘솔](https://console.adobe.io/home)
1. 1단계에서 만든 Adobe ID을 사용하여 로그인
1. 새 프로젝트 만들기

비디오를 따라 Adobe I/O 프로젝트를 생성하십시오.
