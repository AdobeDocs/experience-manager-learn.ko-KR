# 연결된 앱 만들기

연결된 앱은 API 및 표준 프로토콜(예: SAML, OAuth 및 OpenID Connect)을 사용하여 외부 애플리케이션을 Salesforce과 통합할 수 있는 프레임워크입니다. 연결된 앱은 이러한 프로토콜을 사용하여 외부 앱을 인증, 승인 및 제공합니다.
