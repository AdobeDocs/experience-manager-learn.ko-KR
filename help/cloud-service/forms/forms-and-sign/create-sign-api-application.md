# Adobe Sign API 응용 프로그램 만들기

Adobe Sign 플랫폼과 통합하려면 Adobe Sign에서 API 애플리케이션을 만들어야 합니다. API 애플리케이션은 Adobe Sign 플랫폼과 상호 작용하는 클라이언트 애플리케이션의 대표자 역할을 합니다. 고객 또는 파트너 도메인에 API 애플리케이션을 만들 수 있습니다.
