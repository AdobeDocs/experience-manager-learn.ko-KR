+ GraphQL 언어의 강력한 기능 이해
+ 콘텐츠 조각 목록 및 단일 콘텐츠 조각을 쿼리하는 방법에 대해 알아봅니다
+ 특정 데이터 속성을 필터링하고 요청하는 방법을 알아봅니다
