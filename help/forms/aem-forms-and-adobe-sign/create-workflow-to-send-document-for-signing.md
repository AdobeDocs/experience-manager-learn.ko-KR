# 워크플로우를 사용하여 서명을 위해 문서 보내기

서명 문서 구성 요소를 사용하여 AEM 워크플로우를 만듭니다. 서명 구성 요소를 사용하여 서명을 위해 자동 생성된 DoR을 활용하십시오.
