# 제출된 데이터 및 첨부 파일로 양식 미리 채우기

**관리자**&#x200B;가 검색 결과 테이블에서 ViewLink를 클릭하면 양식이 제출된 데이터로 미리 채워집니다.

Azure 포털에서 BlobId와 연결된 데이터를 가져오는 데 미리 채우기 서비스가 사용되었습니다.
