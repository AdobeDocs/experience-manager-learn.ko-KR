# 유형 양식 데이터 모델의 변수

유형 양식 데이터 모델의 변수는 일반적으로 양식 데이터 모델의 기본 데이터 소스에 행을 삽입하는 데 사용됩니다. 이 비디오에서는 AEM 워크플로우를 사용하여 데이터베이스에 행을 삽입하는 데 필요한 단계를 설명합니다.
